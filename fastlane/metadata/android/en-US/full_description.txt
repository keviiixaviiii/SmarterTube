SmarterTube is a phone and tablet YouTube client built on top of the SmartTube TV app (yuliskov/SmartTube). It adds a native portrait-mode UI while leaving all TV functionality intact and staying upstream-mergeable.

<b>What's different from SmartTube TV</b>
• Native phone/portrait layout — Home, Search, Channel, and Settings all open in mobile-friendly screens instead of the TV leanback UI.
• Tablet support — multi-column grid that adapts to sw600dp+ screens with correct landscape/portrait switching.
• Co-installable alongside the SmartTube TV app (separate package: app.smarttube.mobile).

<b>What's the same</b>
• All SmartTube TV features: ad-free playback, SponsorBlock, Return YouTube Dislike, background play, PiP, voice search, subscriptions.
• No Google account required to browse; sign-in is optional and handled through SmartTube's existing flow.
• No Google Play Services required.

<b>Permissions</b>
Zero app-declared permissions beyond what SmartTube TV already ships. Voice search uses the system recognizer intent (runs in Google's process — no RECORD_AUDIO permission required from this app).

<b>Source &amp; attribution</b>
Fork of yuliskov/SmartTube (MIT). All SmartTube TV functionality is unchanged; this fork adds only the phone/tablet UI layer. Tracks upstream SmartTube; this release based on SmartTube 31.88.
